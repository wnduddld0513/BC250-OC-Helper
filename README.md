## Installation

You can easily install the application using the automated installation script.

Run the following command in your terminal:

```bash
sudo curl -sSL https://raw.githubusercontent.com/wnduddld0513/BC250-OC-Helper/main/install.sh | bash
```


CPU governor: https://github.com/bc250-collective/bc250_smu_oc/
GPU governor: https://github.com/filippor/cyan-skillfish-governor/
